# scinewsAI

**ScinewsAI** é uma plataforma Full-Stack projetada para democratizar o acesso à ciência. O sistema automatiza a curadoria de artigos do Arxiv, aplica algoritmos de relevância e utiliza IA Generativa (RAG) para traduzir o "acadêmico" para uma linguagem simples. O resultado é entregue aos usuários através de um **Blog interativo e Newsletter**, permitindo que qualquer pessoa acompanhe as últimas inovações em IA sem barreiras técnicas.
